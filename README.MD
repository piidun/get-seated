# Get Seated!

A game created for Brackeys Game Jam 2026.1.

## Solutions

- HTML5 on Web as target runtime.
- TypeScript as coding language.
  - ImmutableJS for immutable and persistent collections.
- http-server as development server.
- Itch.io as distributor.

## How to Develop

1. Clone repository.
1. Run `npm install` in the root repo to fetch dependencies.
1. Run `npm run start` to start the TypeScript compiler.
1. Run `npm run serve` to start the development server.
   - This should open the page automatically.
1. Refresh the page whenever needed.
